# Pokedex Angular

![build](https://github.com/AkeUS/pokedex-angular/workflows/build/badge.svg?branch=main)

Aplicación desarrollada con Angular

[Pokedex Angular](https://pokedex-angular.akeus.vercel.app)

## Ejecutar entorno local

Clonar el proyecto y ejecutar los siguientes comandos

Descargar dependencias

```shell
npm install
```
Ejecutar aplicación

```shell
npx ng serve
```

## Ejecutar con Docker

```shell
docker run -d -p 3000:80 akeus/pokedex-angular
```

Abrir un navegador e ingresar a http://localhost:3000
